### java Transmitter
   1. javac Transmitter.java
   2. java Transmitter test.txt localhost

java Transmitter <path of file> <ip address> [optional]

Optional:
-h: print help page
-s <integer>: size of data, default: 1472
-p <integer>: port number, default: 1234
